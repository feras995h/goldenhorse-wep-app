Cloudflare Tunnel Setup Instructions

How to Get Your Tunnel URL

Step 1: Open a new Command Prompt or PowerShell window
Step 2: Navigate to your project directory:
cd "c:\Users\Default.DESKTOP-6KVI7UU\Desktop\منضومة وائل v2"
Step 3: Run the Cloudflare tunnel command:
cloudflared tunnel --url http://localhost:3000
Step 4: Look for a line in the output that looks like this:
INFO Your quick Tunnel has been created! Visit it at (it may take some time to be reachable): https://[random-subdomain].trycloudflare.com
Note: With the updated Vite configuration that includes allowedHosts: 'all', your application should now work with any Cloudflare tunnel URL without showing the "Blocked request" error.

Troubleshooting

If you still encounter the "Blocked request" error:

  1. Make sure your vite.config.ts file contains allowedHosts: 'all' in the server configuration
  2. Restart your development server after making any configuration changes
  3. Each time you start a new tunnel, you'll get a different URL which is normal for free Cloudflare tunnels
