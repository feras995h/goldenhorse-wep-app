# ميزة اختيار حساب الأب في دليل الحسابات

## نظرة عامة

تم إضافة ميزة اختيار حساب الأب عند إنشاء أو تعديل الحسابات في دليل الحسابات. هذه الميزة تسمح للمستخدمين بإنشاء هيكل هرمي للحسابات المحاسبية. **الحساب الأب أصبح إجباري** لضمان تنظيم أفضل لدليل الحسابات.

## الميزات المضافة

### 1. حقل اختيار حساب الأب (إجباري)
- **الموقع**: في نموذج إنشاء وتعديل الحسابات
- **الوظيفة**: يسمح للمستخدم باختيار حساب أب للحساب الجديد (إجباري)
- **البحث**: دعم البحث في الحسابات الموجودة بالكود أو الاسم
- **التحقق**: منع اختيار الحساب كأب لنفسه أو الحسابات الفرعية

### 2. تصميم محسن لنافذة إضافة الحساب
- **أقسام منظمة**: تقسيم النموذج إلى أقسام واضحة
- **ألوان مميزة**: كل قسم له لون خلفية مختلف
- **أيقونات بصرية**: نقاط ملونة لتمييز كل قسم
- **تحقق من صحة البيانات**: تعطيل زر الحفظ حتى يتم ملء جميع الحقول المطلوبة

### 3. عرض معلومات الحساب الأب
- **في عرض الحساب**: يظهر الحساب الأب المحدد
- **في شجرة الحسابات**: تمييز بصري للحسابات الرئيسية والفرعية
- **في التصدير**: إضافة عمود الحساب الأب في ملف CSV

### 4. تحسينات شجرة الحسابات
- **أيقونات مميزة**: مجلد للحسابات الرئيسية، ملف للحسابات الفرعية
- **ألوان مميزة**: ألوان مختلفة لكل نوع حساب
- **معلومات إضافية**: عرض عدد الحسابات الفرعية ونوع الحساب

## كيفية الاستخدام

### إنشاء حساب جديد مع حساب أب
1. انقر على "إضافة حساب جديد"
2. املأ المعلومات الأساسية (الكود، الاسم، النوع)
3. **في قسم الحساب الأب**، ابدأ بكتابة كود أو اسم الحساب الأب
4. اختر الحساب الأب من القائمة المنسدلة (إجباري)
5. اختر عملة الحساب
6. احفظ الحساب

### تعديل حساب موجود
1. انقر على الحساب في شجرة الحسابات
2. انقر على "تعديل"
3. يمكنك تغيير الحساب الأب أو إزالته (يجب اختيار حساب أب جديد)
4. احفظ التغييرات

### إزالة حساب الأب
- انقر على زر "إزالة" في قسم الحساب الأب
- **ملاحظة**: يجب اختيار حساب أب جديد قبل الحفظ

## التصميم الجديد

### 1. قسم المعلومات الأساسية (رمادي)
- رقم الحساب (إجباري)
- اسم الحساب (إجباري)
- الاسم بالإنجليزية (اختياري)
- نوع الحساب (إجباري)

### 2. قسم الحساب الأب (أزرق)
- حقل البحث في الحسابات
- عرض الحساب الأب المحدد
- زر إزالة الحساب الأب

### 3. قسم إعدادات العملة (أخضر)
- اختيار عملة الحساب

### 4. أزرار الإجراءات
- زر إلغاء
- زر حفظ (معطل حتى يتم ملء جميع الحقول المطلوبة)

## الدوال الجديدة

### `searchParentAccounts(searchTerm)`
```typescript
const searchParentAccounts = (searchTerm: string) => {
  if (searchTerm.length < 2) {
    setParentAccountResults([]);
    return;
  }

  const results = accounts.filter(account => {
    // Don't show the current account as a parent option when editing
    if (modalMode === 'edit' && selectedAccount && account.id === selectedAccount.id) {
      return false;
    }
    
    // Only show accounts that don't have a parent (main accounts) or have minimal hierarchy
    const isMainAccount = !account.parentId;
    const hasChildren = accounts.some(acc => acc.parentId === account.id);
    
    return (isMainAccount || hasChildren) && (
      account.code.toLowerCase().includes(searchTerm.toLowerCase()) ||
      account.name.toLowerCase().includes(searchTerm.toLowerCase())
    );
  }).slice(0, 10);

  setParentAccountResults(results);
};
```

## الميزات البصرية

### 1. تمييز الحسابات
- **الحسابات الرئيسية**: أيقونة مجلد مع لون مميز
- **الحسابات الفرعية**: أيقونة ملف مع لون مميز
- **أنواع الحسابات**: ألوان مختلفة لكل نوع

### 2. معلومات إضافية
- **عدد الحسابات الفرعية**: عرض عدد الحسابات التابعة
- **نوع الحساب**: شارة ملونة تظهر نوع الحساب
- **حالة الحساب**: مدين/دائن مع الرصيد

### 3. واجهة مستخدم محسنة
- **بحث ذكي**: البحث في الحسابات بالكود أو الاسم
- **قائمة منسدلة**: عرض نتائج البحث مع معلومات إضافية
- **تأكيد الاختيار**: عرض الحساب الأب المحدد مع إمكانية الإزالة
- **تحقق من صحة البيانات**: تعطيل زر الحفظ حتى يتم ملء جميع الحقول

## الفوائد

1. **هيكل هرمي واضح**: تنظيم أفضل للحسابات المحاسبية
2. **سهولة التنقل**: تصفح أسهل في شجرة الحسابات
3. **تصنيف دقيق**: تمييز واضح بين الحسابات الرئيسية والفرعية
4. **تجربة مستخدم محسنة**: واجهة بديهية وسهلة الاستخدام
5. **مرونة في الإدارة**: إمكانية تغيير الحساب الأب
6. **ضمان جودة البيانات**: الحساب الأب إجباري لضمان التنظيم
