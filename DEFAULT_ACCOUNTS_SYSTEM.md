# نظام الحسابات الأساسية التلقائي

## نظرة عامة

تم إضافة نظام الحسابات الأساسية التلقائي إلى دليل الحسابات. هذا النظام يقوم بإنشاء الحسابات الخمسة الأساسية تلقائياً عند بدء النظام لأول مرة، مما يضمن وجود هيكل محاسبي صحيح ومتكامل.

## كيفية عمل النظام

### 1. الإنشاء التلقائي
- عند بدء النظام لأول مرة، يتم التحقق من وجود حسابات
- إذا لم توجد حسابات، يتم إنشاء الحسابات الخمسة الأساسية تلقائياً
- يتم إنشاء كل حساب مع جميع البيانات المطلوبة

### 2. الحماية من التعديل والحذف
- الحسابات الأساسية محمية من التعديل والحذف
- عند محاولة تعديل حساب أساسي، يتم عرض رسالة تنبيه
- عند محاولة حذف حساب أساسي، يتم عرض رسالة تنبيه

### 3. التمييز البصري
- الحسابات الأساسية تظهر مع أيقونة درع (Shield) زرقاء
- في نافذة العرض، تظهر علامة "حساب نظام أساسي"
- في شجرة الحسابات، تظهر أيقونة خاصة

### 3. دالة الإنشاء التلقائي
```typescript
const createDefaultAccounts = async () => {
  try {
    console.log('Creating default accounts...');
    for (const defaultAccount of DEFAULT_ACCOUNTS) {
      await financialAPI.createAccount({
        ...defaultAccount,
        createdAt: new Date().toISOString(),
        updatedAt: new Date().toISOString()
      });
    }
    console.log('Default accounts created successfully');
  } catch (error) {
    console.error('Error creating default accounts:', error);
  }
};
```

## الفوائد

### 1. ضمان الهيكل المحاسبي
- وجود الحسابات الأساسية الخمسة مضمون
- لا يمكن حذف أو تعديل الحسابات الأساسية
- هيكل محاسبي صحيح ومتكامل

### 2. سهولة الاستخدام
- لا حاجة لإضافة الحسابات الأساسية يدوياً
- النظام يعمل تلقائياً من البداية
- واجهة واضحة ومفهومة

### 3. الأمان والاستقرار
- حماية الحسابات الأساسية من التعديل
- منع الأخطاء البشرية
- استقرار النظام المحاسبي

### 4. التمييز البصري
- أيقونات خاصة للحسابات الأساسية
- علامات واضحة في الواجهة
- سهولة التمييز بين الحسابات

## كيفية الاستخدام

### 1. بدء النظام لأول مرة
1. عند تشغيل النظام لأول مرة
2. يتم التحقق من وجود حسابات
3. إذا لم توجد حسابات، يتم إنشاء الحسابات الأساسية تلقائياً
4. يمكن البدء في إضافة الحسابات الفرعية

### 2. إضافة حسابات فرعية
1. انقر على "إضافة حساب جديد"
2. اختر الحساب الأب من الحسابات الأساسية
3. املأ المعلومات المطلوبة
4. احفظ الحساب

### 3. عرض الحسابات الأساسية
1. الحسابات الأساسية تظهر مع أيقونة درع زرقاء
2. عند النقر عليها، تظهر علامة "حساب نظام أساسي"
3. لا تظهر أزرار التعديل والحذف للحسابات الأساسية

## التحسينات المستقبلية

1. **تخصيص الحسابات الأساسية**: إمكانية تخصيص أسماء الحسابات الأساسية
2. **إضافة حسابات أساسية جديدة**: إمكانية إضافة حسابات أساسية إضافية
3. **نسخ الحسابات الأساسية**: إمكانية نسخ هيكل الحسابات الأساسية
4. **تقارير الحسابات الأساسية**: تقارير خاصة بالحسابات الأساسية
5. **تدقيق الحسابات الأساسية**: نظام تدقيق للتأكد من صحة الحسابات الأساسية

## الخلاصة

نظام الحسابات الأساسية التلقائي يضمن وجود هيكل محاسبي صحيح ومتكامل من البداية، مع حماية الحسابات الأساسية من التعديل أو الحذف، مما يوفر استقراراً وأماناً للنظام المحاسبي.
