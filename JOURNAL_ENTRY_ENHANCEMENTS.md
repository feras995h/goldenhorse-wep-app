# تحسينات نظام القيود المحاسبية

## الميزات الجديدة المضافة

### 1. إضافة سطر جديد تلقائياً
- **الوصف**: عند تعبئة بيانات السطر الثاني من القيد، يتم إضافة سطر جديد تلقائياً
- **المنطق**: 
  - يتم فحص السطر الثاني (index 1) عند كل تحديث
  - إذا كان السطر الثاني يحتوي على حساب وبيان أو مبلغ، والسطر الأخير فارغ
  - يتم إضافة سطر جديد تلقائياً
- **الملف**: `client/src/components/Financial/JournalEntryForm.tsx`

### 2. تنظيف الأسطر الفارغة تلقائياً
- **الوصف**: يتم حذف الأسطر الفارغة تلقائياً عند حفظ القيد
- **المنطق**:
  - يتم فحص كل سطر قبل الحفظ
  - السطر يعتبر فارغ إذا لم يحتوي على حساب أو بيان أو مبالغ
  - يتم حذف جميع الأسطر الفارغة قبل إرسال البيانات للخادم
- **الملف**: `client/src/pages/JournalEntries.tsx`

### 3. تمييز بصري للأسطر الفارغة
- **الوصف**: الأسطر الفارغة تظهر بخلفية صفراء للتمييز
- **التنفيذ**: 
  - إضافة class `bg-yellow-50` للأسطر الفارغة
  - يساعد المستخدم على رؤية الأسطر التي تحتاج إلى تعبئة

### 4. زر تنظيف الأسطر الفارغة
- **الوصف**: زر إضافي لتنظيف الأسطر الفارغة يدوياً
- **الموقع**: في شريط الأدوات أعلى جدول القيود
- **الوظيفة**: يزيل جميع الأسطر الفارغة فوراً

## الدوال الجديدة

### `isLineEmpty(line)`
```typescript
const isLineEmpty = (line: JournalEntryLine) => {
  return !line.accountId && 
         !line.description && 
         (!line.debit || line.debit === 0) && 
         (!line.credit || line.credit === 0);
};
```

### `isLineFilled(line)`
```typescript
const isLineFilled = (line: JournalEntryLine) => {
  return line.accountId && 
         (line.description || (line.debit && line.debit > 0) || (line.credit && line.credit > 0));
};
```

### `cleanEmptyLines()`
```typescript
const cleanEmptyLines = () => {
  const cleanedLines = formData.lines.filter(line => !isLineEmpty(line));
  onFormDataChange({ ...formData, lines: cleanedLines });
};
```

## كيفية الاستخدام

1. **إنشاء قيد جديد**:
   - املأ السطر الأول بالحساب والمبلغ
   - املأ السطر الثاني بالحساب والمبلغ
   - سيتم إضافة سطر ثالث تلقائياً

2. **تنظيف الأسطر الفارغة**:
   - استخدم زر "تنظيف الأسطر الفارغة" لإزالة الأسطر غير المستخدمة
   - أو احفظ القيد وسيتم التنظيف تلقائياً

3. **التمييز البصري**:
   - الأسطر الفارغة تظهر بخلفية صفراء
   - الأسطر المملوءة تظهر بخلفية بيضاء
