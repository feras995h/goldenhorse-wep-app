# ميزة الترقيم التلقائي للحسابات

## نظرة عامة

تم إضافة ميزة الترقيم التلقائي للحسابات في دليل الحسابات. هذه الميزة تقوم بإنشاء أرقام الحسابات تلقائياً بناءً على الحساب الأب المحدد، مما يضمن تناسقاً في ترقيم الحسابات وتنظيماً أفضل لدليل الحسابات.

## الميزات المضافة

### 1. الترقيم التلقائي للحسابات الرئيسية
- **النظام**: ترقيم من 1000، 2000، 3000، إلخ
- **المنطق**: البحث عن أعلى رقم موجود وإضافة 1000
- **البداية**: إذا لم توجد حسابات رئيسية، يبدأ من 1000

### 2. الترقيم التلقائي للحسابات الفرعية
- **النظام**: ترقيم هرمي (الأب.001، الأب.002، إلخ)
- **المنطق**: البحث عن أعلى رقم فرعي موجود وإضافة 1
- **التنسيق**: 3 أرقام مع أصفار في البداية (001، 002، إلخ)

### 3. إنشاء الحسابات الرئيسية
- **زر مخصص**: "إنشاء حساب رئيسي"
- **الترقيم التلقائي**: إنشاء رقم تلقائياً للحساب الرئيسي
- **المرونة**: إمكانية إنشاء حساب رئيسي أو فرعي

## كيفية عمل الترقيم

### الحسابات الرئيسية
```
الحساب الأول: 1000
الحساب الثاني: 2000
الحساب الثالث: 3000
... وهكذا
```

### الحسابات الفرعية
```
الحساب الأب: 1000
  - الحساب الفرعي الأول: 1000.001
  - الحساب الفرعي الثاني: 1000.002
  - الحساب الفرعي الثالث: 1000.003

الحساب الأب: 2000
  - الحساب الفرعي الأول: 2000.001
  - الحساب الفرعي الثاني: 2000.002
```

## كيفية الاستخدام

### إنشاء حساب رئيسي
1. انقر على "إضافة حساب جديد"
2. انقر على "إنشاء حساب رئيسي"
3. سيتم إنشاء رقم الحساب تلقائياً (مثال: 1000)
4. املأ باقي المعلومات
5. احفظ الحساب

### إنشاء حساب فرعي
1. انقر على "إضافة حساب جديد"
2. اختر الحساب الأب من القائمة
3. سيتم إنشاء رقم الحساب تلقائياً (مثال: 1000.001)
4. املأ باقي المعلومات
5. احفظ الحساب

### تعديل الحساب الأب
1. عند تغيير الحساب الأب، سيتم تحديث رقم الحساب تلقائياً
2. النظام يمنع تكرار أرقام الحسابات

## الدوال الجديدة

### `generateAccountCode(parentAccount)`
```typescript
const generateAccountCode = (parentAccount: Account) => {
  // Get all child accounts of the selected parent
  const childAccounts = accounts.filter(acc => acc.parentId === parentAccount.id);
  
  if (childAccounts.length === 0) {
    // If no children, create first child code
    const newCode = `${parentAccount.code}.001`;
    setFormData(prev => ({ ...prev, code: newCode }));
  } else {
    // Find the highest existing child code
    const existingCodes = childAccounts.map(acc => {
      const parts = acc.code.split('.');
      return parts.length > 1 ? parseInt(parts[parts.length - 1]) : 0;
    });
    
    const maxCode = Math.max(...existingCodes);
    const nextCode = maxCode + 1;
    const newCode = `${parentAccount.code}.${nextCode.toString().padStart(3, '0')}`;
    setFormData(prev => ({ ...prev, code: newCode }));
  }
};
```

### `handleCreateMainAccount()`
```typescript
const handleCreateMainAccount = () => {
  const mainAccounts = accounts.filter(acc => !acc.parentId);
  let newCode = '1000';
  
  if (mainAccounts.length > 0) {
    const existingCodes = mainAccounts.map(acc => {
      const codeNum = parseInt(acc.code) || 0;
      return codeNum;
    });
    const maxCode = Math.max(...existingCodes);
    newCode = (maxCode + 1000).toString();
  }
  
  setFormData(prev => ({ 
    ...prev, 
    code: newCode,
    parentId: '' 
  }));
  setSelectedParentAccount(null);
  setParentAccountSearch('');
};
```

## الميزات البصرية

### 1. عرض الرقم المقترح
- **للحسابات الرئيسية**: عرض الرقم المقترح في قسم أصفر
- **للحسابات الفرعية**: عرض الرقم المقترح في قسم الحساب الأب المحدد

### 2. زر إنشاء حساب رئيسي
- **الموقع**: في قسم الحساب الأب
- **الوظيفة**: إنشاء حساب رئيسي مع ترقيم تلقائي
- **التصميم**: زر أزرق مميز

### 3. تحديث تلقائي للرقم
- **عند اختيار الحساب الأب**: تحديث الرقم تلقائياً
- **عند إنشاء حساب رئيسي**: تحديث الرقم تلقائياً

## الفوائد

1. **تناسق في الترقيم**: نظام ترقيم موحد ومنظم
2. **توفير الوقت**: لا حاجة لإدخال الأرقام يدوياً
3. **منع الأخطاء**: تجنب تكرار أرقام الحسابات
4. **تنظيم هرمي**: ترقيم يعكس هيكل الحسابات
5. **سهولة الاستخدام**: واجهة بديهية لإنشاء الحسابات

## التحقق من صحة البيانات

### 1. منع تكرار الأرقام
- التحقق من وجود الرقم قبل الحفظ
- رسالة تنبيه عند تكرار الرقم

### 2. التحقق من الحساب الأب
- للحسابات الفرعية: يجب اختيار حساب أب
- للحسابات الرئيسية: لا حاجة لحساب أب

### 3. التحقق من الحقول المطلوبة
- رقم الحساب (إجباري)
- اسم الحساب (إجباري)
- نوع الحساب (إجباري)
